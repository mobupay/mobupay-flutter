# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability in this Mobupay integration or
in the Mobupay API, please report it privately to **support@mobupay.nc**.

Please do NOT open a public issue for security reports.

We will acknowledge your report within 72 hours and keep you informed of the
remediation progress. Coordinated disclosure is appreciated: give us a reasonable
time to ship a fix before any public disclosure.

## Scope

- This repository (integration code, SDK).
- The Mobupay API (`api.mobupay.nc`) and hosted payment pages.

## Supported versions

Only the latest released version of this integration receives security fixes.
Merchants are strongly encouraged to keep the integration up to date.
